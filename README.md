В данном проекте представлены демонстрационные автотесты.  
Тесты написанный на **Selenium** Java.  
Тесты запускаются локально.  

### Автотесты для [**_Swag Labs_**](https://www.saucedemo.com):
<details> <summary>1. Авторизация (login) </summary> 

LoginTest:  
- Проверка страницы авторизации
- Вход с валидными данными
- Вход с заблокированным пользователем
- Вход с пустым пользователем
- Вход с пустым паролем
- Вход с неверным пользователем
- Вход с неверным паролем

LogoutTest:  
- Logout
</details>

<details>
<summary>2. Главная страница (mainpage)</summary>   

MainPageSortTest:    
- Сортировка товаров по алфавиту
- Сортировка товаров по алфавиту в обратном порядке
- Сортировка по возростанию цены
- Сортировка по убыванию цены

MainPageTest:
- Проверка открытия страницы
- Проверка наличия карточек
- Проверка содержимого карточки
</details>

<details>
<summary>3. Карточка товара (card)</summary>   

CardTest:   
- Проверка содержимого в карточке
</details>

<details>
<summary>4. Корзина (cart)</summary>   

CartTest:
- Проверка корзины на отстуствие товара
- Проверка корзины на присутствие товара
</details>
